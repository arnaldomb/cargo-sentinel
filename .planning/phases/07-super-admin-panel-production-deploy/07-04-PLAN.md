    O script aceita variáveis de ambiente opcionais:
    - `API_URL` (default: `http://localhost:4000`)
    - `DB_URL` (default: `postgresql://sentinel:sentinel@localhost:5432/cargo_sentinel`)

    Estrutura do script:

    ```bash
    #!/usr/bin/env bash
    set -euo pipefail

    API_URL="${API_URL:-http://localhost:4000}"
    DB_URL="${DB_URL:-postgresql://sentinel:sentinel@localhost:5432/cargo_sentinel}"
    PLATE="SMOKE$(date +%s)"

    pass() { echo "[PASS] $1"; }
    fail() { echo "[FAIL] $1"; exit 1; }

    echo "============================================"
    echo " Cargo Sentinel — Smoke Test"
    echo " API: $API_URL"
    echo "============================================"

    # ETAPA 1: Health check
    echo ""
    echo "--- Etapa 1: Health endpoint ---"
    STATUS=$(curl -s -o /dev/null -w "%{http_code}" "$API_URL/api/health")
    [ "$STATUS" = "200" ] && pass "GET /api/health → 200" || fail "GET /api/health → $STATUS (esperado 200)"

    # ETAPA 2: Webhook LPR mock (câmera Intelbras)
    echo ""
    echo "--- Etapa 2: Webhook LPR mock ---"
    # Payload mínimo com base64 de 1x1 pixel PNG transparente
    TINY_IMAGE="iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAYAAAAfFcSJAAAADUlEQVR42mNkYPhfDwAChwGA60e6kgAAAABJRU5ErkJggg=="
    PAYLOAD=$(printf '{"plateNumber":"%s","cameraCode":"LPR-SMOKE-01","direction":"ENTRADA","capturedAt":"%s","imageBase64":"%s"}' \
      "$PLATE" "$(date -u +%Y-%m-%dT%H:%M:%SZ)" "$TINY_IMAGE")
    LPR_STATUS=$(curl -s -o /dev/null -w "%{http_code}" \
      -X POST "$API_URL/api/lpr/NotificationInfo/vehicle" \
      -H "Content-Type: application/json" \
      -d "$PAYLOAD")
    [ "$LPR_STATUS" = "200" ] && pass "POST /api/lpr/NotificationInfo/vehicle → 200" || fail "LPR webhook → $LPR_STATUS (esperado 200)"

    # ETAPA 3: Aguardar processamento assíncrono (BullMQ worker)
    echo ""
    echo "--- Etapa 3: Aguardar processamento assíncrono (max 10s) ---"
    FOUND=0
    for i in $(seq 1 10); do
      sleep 1
      COUNT=$(psql "$DB_URL" -t -c "SELECT COUNT(*) FROM \"Evento\" WHERE \"placaNumero\" = '$PLATE';" 2>/dev/null | tr -d ' ')
      if [ "${COUNT:-0}" -gt "0" ]; then
        FOUND=1
        break
      fi
    done
    [ "$FOUND" = "1" ] && pass "Evento com placa $PLATE encontrado no banco após processamento" \
      || fail "Evento com placa $PLATE NÃO encontrado no banco após 10s — verifique o worker BullMQ"

    # ETAPA 4: Idempotência — reenviar mesmo payload não duplica
    echo ""
    echo "--- Etapa 4: Idempotência (reenvio não duplica) ---"
    curl -s -o /dev/null -X POST "$API_URL/api/lpr/NotificationInfo/vehicle" \
      -H "Content-Type: application/json" \
      -d "$PAYLOAD"
    sleep 2
    COUNT2=$(psql "$DB_URL" -t -c "SELECT COUNT(*) FROM \"Evento\" WHERE \"placaNumero\" = '$PLATE';" 2>/dev/null | tr -d ' ')
    [ "${COUNT2:-0}" = "1" ] && pass "Idempotência OK — $PLATE tem exatamente 1 registro após 2 envios" \
      || fail "Idempotência FALHOU — $PLATE tem $COUNT2 registros (esperado 1)"

    echo ""
    echo "============================================"
    echo " SMOKE TEST CONCLUÍDO COM SUCESSO"
    echo "============================================"
    ```

    # Cargo Sentinel

    - **Turborepo + pnpm** (monorepo)

    ## Deploy em Produção (Hostinger VPS)

    ### Pré-requisitos

    - VPS Ubuntu 22.04+ com no mínimo 2 vCPUs e 4 GB RAM
    - Docker Engine 24+ e Docker Compose Plugin instalados
    - Domínio apontado para o IP da VPS (registro A configurado)
    - Portas 80 e 443 abertas no firewall

    ### Passo a passo

    #### 1. Clonar o repositório

    ```bash
    git clone <url-do-repo> cargo-sentinel
    cd cargo-sentinel
    ```

    #### 2. Configurar variáveis de ambiente

    ```bash
    cp .env.example .env
    nano .env   # Preencher todos os valores (ver comentários no arquivo)
    ```

    Variáveis obrigatórias:
    | Variável | Descrição |
    |----------|-----------|
    | `AUTH_SECRET` | Segredo JWT — gerar com `openssl rand -base64 32` |
    | `PUBLIC_DOMAIN` | Domínio sem https:// (ex: app.seudominio.com.br) |
    | `ACME_EMAIL` | Email para certificados Let's Encrypt |
    | `POSTGRES_PASSWORD` | Senha do banco de dados |
    | `GARAGE_ACCESS_KEY` / `GARAGE_SECRET_KEY` | Credenciais do object storage |
    | `GARAGE_SERVER_URL` | URL pública HTTPS do Garage (ex: https://media.seudominio.com.br) |
    | `EVOLUTION_API_KEY` | Chave da Evolution API WhatsApp |

    #### 3. Preparar volumes persistentes

    ```bash
    # Certificado Let's Encrypt (deve existir antes do primeiro up)
    mkdir -p traefik
    touch traefik/acme.json
    chmod 600 traefik/acme.json

    # Configuração do Garage
    mkdir -p garage
    # Copiar garage.toml de exemplo (ver garage/garage.toml.example se existir)
    ```

    #### 4. Build e subida dos serviços

    ```bash
    # Produção: usa apenas docker-compose.yml (sem override de dev)
    docker compose -f docker-compose.yml build

    docker compose -f docker-compose.yml up -d
    ```

    #### 5. Migração do banco e seed inicial

    ```bash
    # Executar migration + seed uma única vez
    docker compose -f docker-compose.yml run --rm \
      -e DATABASE_URL="${DATABASE_URL}" \
      api sh -c "pnpm --filter @cargo-sentinel/database run db:push && pnpm --filter @cargo-sentinel/database run seed"
    ```

    #### 6. Verificar status dos serviços

    ```bash
    docker compose -f docker-compose.yml ps

    # Todos os serviços devem estar: Up (healthy)
    # Traefik emite certificado Let's Encrypt automaticamente na primeira requisição HTTPS
    ```

    #### 7. Smoke test de produção

    ```bash
    # Instalar psql client se necessário: apt install postgresql-client

    API_URL="https://app.seudominio.com.br" \
    ```

    ### Troubleshooting

    **Traefik não emite certificado:**
    - Verificar que `traefik/acme.json` existe e tem permissão 600
    - Verificar que o domínio está apontado para o IP da VPS
    - Ver logs: `docker compose logs traefik`

    **API não inicia:**
    - Verificar variáveis obrigatórias: `docker compose logs api | head -20`
    - Erro "Variáveis de ambiente obrigatórias não configuradas" → revisar `.env`

    **Garage não aceita uploads:**
