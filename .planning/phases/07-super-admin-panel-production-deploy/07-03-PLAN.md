<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HOST → Docker socket | Traefik monta /var/run/docker.sock:ro — risco aceito em VPS single-tenant |
| Internet → :80 | Traefik redireciona para :443 — HTTP não serve conteúdo |
| Internet → :443 | Traefik termina TLS com Let's Encrypt antes de rotear |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-11 | Denial of Service | Serviço sem restart policy | mitigate | `restart: unless-stopped` em todos os 7 serviços — container reinicia automaticamente após crash |
| T-07-12 | Denial of Service | Serviço dependente sobe antes do dependency | mitigate | `condition: service_healthy` no depends_on garante ordem de startup correta |
| T-07-13 | Information Disclosure | .env.example com segredos reais | mitigate | .env.example contém APENAS placeholders (TROCAR_AQUI) — nunca valores reais; .env está no .gitignore |
| T-07-14 | Denial of Service | API inicia sem AUTH_SECRET | mitigate | Fail-fast via `process.exit(1)` antes de aceitar conexões — não sobe em estado inválido |
| T-07-15 | Spoofing | Certificado Let's Encrypt | accept | TLS Challenge (ACME) verifica controle do domínio antes de emitir certificado |
</threat_model>
